# `<elgg-listing-table>`

## Props

 * `items` - **[required]** An array of items
 * `options` - List rendering options
 
## Usage

```html
<elgg-listing-table
    :items="[]"
    :options="{
       columns: [], // see <elgg-table>
       class: ['custom-css-class'],
    }"
></elgg-listing-table>
```
