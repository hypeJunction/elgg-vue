# `<elgg-icon>`

## Props

 * `name` - **[required]** icon name
 * `size` - icon size
 * `color` - icon color (e.g. `danger`)
 * `state` - icon state (e.g. `selected`)
 * `position` - `left` or `right`
 
## Usage

```html
<elgg-icon
    name="ban"
    color="danger"
    size="small"
    state="active"
    position="left"
></elgg-icon>
```
