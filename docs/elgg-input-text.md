# `<elgg-input-text>`

## Props

 * `data-name`
 * `data-value`
 * `data-label`
 * `data-help`
 * `data-placeholder`
 * `data-required`
  
## Usage

```html
<elgg-input-text
    v-model="value"
    :data-id="id"
    :data-name="name"
    :data-label="Label"
    :data-help="Help text"
    :data-placeholder="Placeholder"
    :data-required="Required"
></elgg-list>
```
