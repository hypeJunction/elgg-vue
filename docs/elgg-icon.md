# `<elgg-icon>`

## Props

 * `data-name` - icon name
 * `data-size` - icon size
 * `data-style` - icon style (e.g. `danger`)
 * `data-state` - icon state (e.g. `selected`)
 
## Usage

```html
<elgg-icon
    data-name="ban"
    data-style="danger"
    data-size="small"
    data-state="active"
></elgg-icon>
```
